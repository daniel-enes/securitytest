It's necessary create a .env file in the root directory with this variable:

JWT_SECRET_KEY=

After, you can access https://randomkeygen.com/ and generate a new secret random key.

Cop the field on "256-bit WEP Keys" and past together with JWT_SECRET_KEY. Example:

JWT_SECRET_KEY=49E8521B93EDFBC4466EE28F4E275AB3E27F169E43E79591C5991DD295
